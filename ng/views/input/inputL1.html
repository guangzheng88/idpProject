组件标签：
    <lj-input-l-1  msg='msgj' active='ljActivej' value='valuej' defaultmsg="请输入6-15位密码"></lj-input-l-1>
组件说明：
    1.msg 变量: 错误提示信息
    2.active 变量:错误样式 active true 显示正确样式 false 显示错误样式
    3.value 变量:绑定input 框输入的值
    4:defaultmsg input框默认提示信息，可以自行设置，也可以设置为空
    5:该input框适用于单个的密码框
