蓝景丽家组件说明

1、此组件库 是在结合 Bootstrap 和 Angularjs 的基础上而开发
2、Bootstrap 的所有 JavaScript 插件都依赖 jQuery,因此 jQuery 必须在 bootstrap.min.js
之前引入;所以,在开发过程中涉及到有引用以上两个文件的位置，您不能擅自修改 3、Bootstrap 对 jQuery 有版本要求, 建议引入 jQuery1.9.1 以上版本, 本组件库目前引入的版本 为1.11.3,相对来说比较稳定,如没有特殊情况,不建议更改
4、当您把下载的 LJLJCL 压缩包进行解压后,将获得两个文件夹,分别为:commom和ljApp;common为组件相 关核心代码,无需关注;您的开发环境或者测试环境为:ljApp
5、您可以在 ljApp/js/app.js 文件中设置路由,在 ljApp/js/controller.js 中写相关业务逻辑,在 ljApp/views 文件夹中创建页面
